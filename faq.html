Frequently Asked Questions

What is this site?

This website aims to be the definitive resource for PSP emulation on PS3.
Our focus is "PSP HD & Minis", the software emulator available for all PS3 models.

Where does the data come from?

Compatibility info is currently obtained from the PS3 Developer Wiki compatibility list.

Will this work on any PS3?

Yes. Sony developed a PSP emulator called "pspemu". However, it is significantly less developed than the PS1 and PS2 emulators. Technical information and comparisons are also available on PS3 Developer Wiki.

How can I contribute?

If a game works perfectly, you may edit the Wiki directly.
If you test an untested game and discover it has issues, add it to the Wiki and open an issue via the GitHub repository.
